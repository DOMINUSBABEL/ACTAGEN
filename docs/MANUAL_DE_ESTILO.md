# MANUAL DE ESTILO Y REDACCIÓN - CONCEJO DE MEDELLÍN
> **Agent Config ID:** STYLE_V3_2026
> **Strictness Level:** 10/10

## 1. PUNTUACIÓN Y COMILLAS
- **Regla Oro:** Primer nivel con comillas Inglesas (“”), segundo nivel con comillas españolas «».
- **Prohibición:** El uso de comillas simples ('') resultará en un flag de error crítico.
- **Puntuación:** El punto (.) siempre se sitúa posterior a cualquier signo de cierre (comillas, paréntesis, corchetes).

## 2. CIFRAS Y MONEDA (Configuración de Formateo)
- **Moneda:** `$ [espacio] [valor]`. Ejemplo: `$ 500`.
- **Escalas:**
  - 4 cifras: Sin separación (`$ 3450`).
  - 5+ cifras: Separación por espacio (`$ 13 450`).
  - Grandes valores: `$ 1 571 682 355`.
- **Nomenclatura:** Si se usa la palabra "billones", se omite la palabra "pesos" para evitar redundancia legislativa.

## 3. CARGOS Y ENTIDADES (Case Sensitivity)
- **Minúsculas (Cargos):** alcalde, secretario, concejal, personero, contralor.
- **Mayúsculas (Instituciones):** Secretaría de Educación, Concejo de Medellín.
- **Siglas:** 
  - < 5 letras: Mayúscula sostenida (CCCP).
  - >= 5 letras: Solo inicial (Inder, Dagrd).

## 4. ESTRUCTURA AGÉNTICA DE INTERVENCIÓN
- **Formato Estándar:** `Intervino el [cargo], [Nombre Completo]:`
- **Marcas de Audio:** 
  - Ininteligible -> `(sic)`
  - Pausas largas -> `[...]`
- **Votaciones:** Lista numerada de concejales con sentido de voto explícito.

## 5. REVISIÓN DE FORMATO
- **Imágenes:** Propiedad `BehindText` (detrás del texto) y alineación `Centered`.
- **Citas:** Sangría francesa de 2cm para textos que superen las 3 líneas.
